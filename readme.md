## Controls

- **Arrow Keys** - Move / Jump
- **Up** - Jump / Hold Up to Reflect off walls. 
- **Tilde/Tick** (\`) - Open Console. Type `help` for cheat codes.

Alternatively, you can plug in a gamepad and it will automatically be recognized.
